# Переменные и константы

### Переменные

Переменные служат для хранения значений.

```java
double sum;
int x = 23;
boolean isLong = true;
char ch = 'a';
float fl = 34.2;
```

### Константы

Константы являются переменными значения которых не изменяются.

```java
// внутри метода
final double CM_PER_INCH = 2.54;

// на уровне класса
public static double CM_PER_INCH = 2.54; 
```
